Zero Bullshit Setup

One binary. No server-side plugins. No dependency hell. No auth configuration that requires a PhD.

jsn setup # Interactive first-time setup
jsn tables # Start exploring

Agent-Native

Designed for AI agents. JSON output, breadcrumbs, and machine-readable structure. Works with Claude, Codex, Cursor, and more.

jsn tables --json # Structured output
jsn tables --agent # Agent mode (no prompts)

Explore Your Instance

Understand your ServiceNow instance without clicking through 12 UI screens. Tables, schemas, business rules, flows, and more.

jsn tables schema incident # Table structure
jsn rules --table incident # Business rules
jsn flows # Active flows

Quick Setup for AI Agents

1. Install CLI

curl -fsSL https://jsn.jace.pro/install | bash
jsn --version # Verify: jsn version X.Y.Z

2. Authenticate

jsn setup # Interactive setup
jsn auth status # Verify: Authenticated

3. Install Skill

npx skills add jacebenson/jsn # One command, works with all agents
npx skills list # Verify servicenow skill is installed

4. Start Using

# Restart your agent to load the skill
# Then just ask naturally about ServiceNow:
# "What tables exist?" or "List business rules on incident"
# The agent automatically invokes jsn commands for you
